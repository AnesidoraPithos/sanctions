# Entity Background Research - Frontend

Next.js 14+ frontend for the Entity Background Research Agent.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Technology Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Axios

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Project Structure

- `app/` - Pages and layouts
- `components/` - React components
- `lib/` - API client and types

## Backend Dependency

Requires FastAPI backend running on port 8000.
